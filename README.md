# Italia’s Pizzeria Website

## Project Description

This is a simple three-page website for a fictional restaurant called **Italia’s Pizzeria**. The site
includes a homepage, a menu page, and a contact page. The purpose of the project is to practice
HTML, CSS, wireframing, and GitHub workflows.
---

## User Stories

1. **As a customer, I want to browse the pizza menu so that I can choose what I want to order.**
2. **As a customer, I want to easily find the restaurant’s phone number so I can quickly place an
order.**
3. **As a customer, I want to view the restaurant’s hours and address so I know when and where
to pick up my food.**
---
